## Обязательный функционал (Must Have) — 50 баллов

### 1. Вставка таблицы MWS Tables в страницу
- Slash-команда `/table` → модалка выбора таблицы
- Embed компонент с live-данными из API
- Таблица редактируется inline (не ссылка, а живой объект)

### 2. In-line автосохранение
- Локальный кэш (IndexedDB/localStorage)
- Debounced sync с бэкендом (1-2 сек)
- Индикатор статуса: "Сохранено" / "Сохраняется..."

### 3. Backlinks (обратные ссылки)
- При упоминании страницы `[[Page Name]]` создаётся связь
- На каждой странице блок "Ссылаются на эту страницу"
- Клик → переход

### 4. Slash-menu с горячими клавишами
- `/` открывает меню команд
- Команды: heading, list, table, link, divider, etc.
- Горячие клавиши: `Ctrl+B`, `Ctrl+I`, `Ctrl+K` (ссылка)

### 5. Совместная работа
- Realtime cursors (видно кто где редактирует)
- CRDT через Yjs (конфликты невозможны)
- Presence: аватары онлайн-пользователей
