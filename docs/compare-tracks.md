## Трек 1: LocalScript — Локальная агентская система для генерации Lua-кода (платформа MWS Octapi)

**Задача:** Разработай автономную агентскую систему на локальной (лёгкой) LLM, которая генерирует и валидирует Lua-код без отправки данных во внешние сервисы.

**Ключевые требования:**
- Локальная LLM (ollama, llama.cpp, Phi-3, Qwen 2.5-Coder и т.д.)
- Генерация Lua-кода по запросу
- Валидация сгенерированного кода (AST-парсинг, синтаксическая проверка)
- Без отправки данных наружу (privacy-first, offline)
- Интеграция с платформой MWS Octapi

**Вероятный стек:** Python + FastAPI/CLI + ollama/llama.cpp + luaparse/lupa + Rich/TUI

---

## Трек 2: WikiLive — Живые таблицы в тексте (платформа MWS Tables)

**Задача:** Создай модуль живой вики-системы, где текст и таблицы становятся единым инструментом для совместной работы и управления знаниями.

**Ключевые требования:**
- Wiki-синтаксис для текста
- Живые/интерактивные таблицы (editable, sortable, filterable)
- Real-time collaboration (несколько пользователей одновременно)
- Единое пространство текста + таблицы
- Интеграция с платформой MWS Tables

**Вероятный стек:** Next.js + TypeScript + Tailwind + TipTap (rich text) + Yjs/Hocuspocus (WebSocket/CRDT)
